# Core deps. Most of these install fine via plain pip from PyPI.
#
# IMPORTANT: torch + torchvision must be installed from the matching CUDA
# wheel index, otherwise pip will silently grab the CPU-only torchvision
# wheel and `torchvision::nms` will fail on CUDA tensors at validation
# time (manifests as a NotImplementedError mid-training). Install with:
#
#   pip install --index-url https://download.pytorch.org/whl/cu118 \
#       torch==2.4.1 torchvision==0.19.1
#
# (or substitute cu121 / cpu / etc. to match your local CUDA toolkit).
